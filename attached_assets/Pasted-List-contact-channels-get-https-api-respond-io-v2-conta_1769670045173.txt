List contact channels
get
https://api.respond.io/v2/contact/{identifier}/channels

List all connected channel of a specific contact.
Request
Path Parameters
identifier
string
required

Identifier of the contact that can either be a phone number, email or contact ID. Examples: id:123 , email:abdc@gmail.com , phone:+60121233112
Query Parameters
cursorId
integer

Pointer to the contact id from where the next batch of records should start from.
limit
integer

Number of contacts to return i.e. page size.
>= 1< 100
Default:
10
